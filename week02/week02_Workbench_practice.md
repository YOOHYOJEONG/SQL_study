# Week 02 Workbench 실습
---
## Workbench의 장점
- 이름을 입력하고 조건을 클릭 몇 번으로 지정가능.
- 테이블과 관계 다이어그램을 자동으로 그려 줌.
- SQL DDL 파일을 export 한 뒤 실행하여 바로 테이블 생성 가능.
